# Descripción del Curso
%-----------------------------------------------------------------------------------------------------
(Última actualización: 26 Agosto, 2025)

Este curso de 8 módulos está diseñado para profesionales y publico en general que desean adquirir nuevas capacidades en el ámbito de Ciencia de Datos, y les proporciona las competencias necesarias para realizar análisis estadísticos rigurosos utilizando Python, integrando herramientas de Inteligencia Artificial en cada etapa del proceso.


# Objetivos
%-----------------------------------------------------------------------------------------------------
## General
Desarrollar aplicaciones para analizar y modelar datos utilizando el lenguaje de programación Python, con el fin de evaluar patrones, tendencias y realizar predicciones.

## Específicos
- Introducción a la computación en la nube (Google Colaboratory).
- Establecer bases en el conocimiento de programación básica emplenado el lenguaje de programación Python.
- Procesar, analizar y construir base de datos provenientes de archivos en formato CSV y XLSX.
- Emplear paquetes o librerias estándar de Python para generar gráficos descriptivos que permitan visualizar tendencias y patrones en datos procesados.
- Organizar y resumir datos para realizar pruebas de normalidad, hipótesis paramétricas y no paramétricas.
- Implementar modelos de regresión lineal simple. 

# Metodología
%-----------------------------------------------------------------------------------------------------
En este curso se utilizará una combinación de clases magistrales, debates en clase, presentaciones, videos, aprendizaje cooperativo y aprendizaje basado en problemas. Este curso esta diseñado para ser desarrollado en 8 sesiones, cada sesión con una duración de 5 horas. A lo largo del desarrollo de cada sesión, aprenderás de manera práctica los siguientes temas:

**MODULO 1 : Introducción a la Programación Básica usando Python**
Este módulo establece las bases del pensamiento computacional y la programación en Python, el lenguaje estándar en
ciencia de datos. Los estudiantes aprenderán a instalar y configurar el entorno de trabajo (Google Colab), dominarán la
sintaxis básica (variables, tipos de datos, estructuras de control), funciones y manejo de estructuras de datos fundamentales
(Listas, Arrays y Data Frames). Se introducirán las librerías esenciales como NumPy para cálculo numérico y Pandas para
manipulación de datos, preparando el terreno para análisis estadísticos posteriores. No se requiere experiencia previa en
programación.

**MODULO 2: Procesamiento de Datos**
Enfocado en la preparación de datos para el análisis estadístico, este módulo aborda el ciclo completo de limpieza y
transformación de datos. Los estudiantes aprenderán a importar datos desde múltiples fuentes (CSV y Excel), identificar y
tratar valores faltantes, detectar y corregir outliers, realizar transformaciones de variables (normalización, estandarización,
codificación de variables categóricas) y crear pipelines de preprocesamiento reutilizables. Se explorarán técnicas de
ingeniería de características básicas y la importancia de la calidad de datos en la investigación científica, utilizando
Numpy, Pandas y Scikit-learn.

**MODULO 3: Visualización de Gráficos**
Este módulo desarrolla la capacidad de comunicar hallazgos estadísticos de manera efectiva a través de visualizaciones. Se
cubrirán las principales librerías de Python: Matplotlib para gráficos personalizables, Seaborn para visualizaciones
estadísticas elegantes. Los estudiantes aprenderán a seleccionar el tipo de gráfico apropiado según la naturaleza de los
datos y la pregunta de investigación (histogramas, boxplots, scatter plots, heatmaps de correlación, gráficos de
distribución), aplicar principios de percepción visual y diseño narrativos que sustenten la toma de decisiones basada en
evidencia.

**MODULO 4: Estadística Descriptiva**
Aquí se fundamentan los conceptos estadísticos esenciales para caracterizar conjuntos de datos. Se estudiarán las medidas
de tendencia central (media, mediana, moda), medidas de dispersión (varianza, desviación estándar, rango intercuartílico),
medidas de forma (asimetría, curtosis) y medidas de posición (percentiles y cuartiles). Los estudiantes aprenderán a
calcular e interpretar tablas de frecuencias, distribuciones de probabilidad empíricas, y a generar resúmenes estadísticos
completos usando Python. Se enfatizará la diferenciación entre estadísticos muestrales y poblacionales, y su relevancia en
el diseño de investigación.

**MODULO 5: Estadística Inferencial**
Este módulo introduce el razonamiento inferencial para generalizar conclusiones desde muestras hacia poblaciones. Se
cubrirán los conceptos de muestreo, distribuciones muestrales, teorema central del límite, intervalos de confianza, y
pruebas de hipótesis (paramétricas y no paramétricas). Los estudiantes implementarán en Python pruebas t para una y dos
muestras, ANOVA de un factor, pruebas de chi-cuadrado, y pruebas de normalidad. Se discutirán los errores Tipo I y Tipo
II, el valor p, el tamaño del efecto y el poder estadístico, con aplicaciones a problemas de investigación reales en diversas
disciplinas profesionales.

**MODULO 6: Implementar Modelos de Regresión Lineal Simple**
Este módulo inicia el estudio del modelado predictivo con el modelo de regresión lineal simple (una variable
independiente). Se abordarán los supuestos del modelo, estimación de coeficientes por mínimos cuadrados ordinarios,
interpretación de parámetros (pendiente e intercepto), coeficiente de determinación (R²), y análisis de residuos para validar
supuestos. Los estudiantes implementarán el modelo usando Statsmodels (enfoque estadístico con inferencia) y Scikit-
learn (enfoque predictivo).

**MODULO 7: Implementar Modelos de Regresión Lineal Multivariado**
Se extienden los conceptos del módulo anterior a escenarios con múltiples variables predictoras. Se estudiarán la matriz de
correlaciones, multicolinealidad y sus diagnósticos (VIF), selección de variables (métodos stepwise, regularización Ridge
y Lasso), interacciones entre variables, y transformaciones polinómicas. Los estudiantes aprenderán a construir, entrenar e
interpretar modelos multivariados complejos, evaluar la significancia individual y conjunta de predictores, y utilizar
técnicas de validación cruzada. Se integrarán herramientas de IA para la optimización de hiperparámetros y la generación
automática de features.

**MODULO 8: Evaluación de Modelos**
El módulo final consolida las competencias en validación y comparación de modelos estadísticos. Se cubrirán métricas de
rendimiento para regresión (MAE, MSE, RMSE, MAPE, R² ajustado), técnicas de validación (hold-out, k-fold cross-
validation, leave-one-out), análisis de sesgo-varianza, y métodos de comparación de modelos. Los estudiantes aprenderán a
detectar sobreajuste (overfitting) y subajuste (underfitting), aplicar técnicas de ensemble básicas, y comunicar resultados
de manera rigurosa.

<script>
(function() {
  let languagesSlideIndex = 0;
  let languagesAutoPlayInterval = null;
  const languagesSlides = document.getElementsByClassName("languages-slides");
  const languagesPlayBtn = document.getElementById("languages-play-btn");

  function languagesShowSlide(index) {
    if (index >= languagesSlides.length) languagesSlideIndex = 0;
    if (index < 0) languagesSlideIndex = languagesSlides.length - 1;
    for (let i = 0; i < languagesSlides.length; i++) {
      languagesSlides[i].classList.remove("active");
    }
    languagesSlides[languagesSlideIndex].classList.add("active");
  }

  window.languagesChangeSlide = function(n) {
    languagesSlideIndex += n;
    languagesShowSlide(languagesSlideIndex);
  };

  window.languagesToggleAutoPlay = function() {
    if (languagesAutoPlayInterval) {
      clearInterval(languagesAutoPlayInterval);
      languagesAutoPlayInterval = null;
      languagesPlayBtn.textContent = "Play";
    } else {
      languagesAutoPlayInterval = setInterval(() => {
        languagesSlideIndex++;
        languagesShowSlide(languagesSlideIndex);
      }, 3000); // Cambia cada 3 segundos
      languagesPlayBtn.textContent = "Pause";
    }
  };

  languagesShowSlide(languagesSlideIndex);
})();
</script>
